Contexte du Projet

Le projet MainsduCoeur est un site web vitrine dédié à la collecte de fonds en faveur des orphelins. Elle permet aux donateurs de soutenir facilement des projets à caractère humanitaire et éducatif, offrant ainsi une aide concrète aux enfants dans le besoin.

Fonctionnalités Principales

  • Présenter clairement les raisons de chaque collecte
  • Faciliter les dons grâce à un système de paiement intégré via FedaPay
  • Permettre le partage rapide des cagnottes sur les réseaux sociaux
  • Offrir un moyen de contact direct avec l’organisateur
  • Garantir une compatibilité sur tous les types d’appareils (mobile, tablette, desktop)

Mon Rôle

  • Réalisation de l’analyse fonctionnelle à partir des besoins du client
  • Développement de l’interface utilisateur avec HTML, CSS et JavaScript
  • Intégration de l’API FedaPay pour le paiement en ligne sécurisé
  • Réalisation de tests unitaires et validation des principales fonctionnalités
  • Déploiement et mise en production du site web

Apprentissages

Ce projet m’a permis d’approfondir mes compétences en développement front-end, notamment dans la gestion de la structure et du design d’une interface utilisateur avec des styles harmonieux et accessibles. Il m’a également permis de découvrir et d’appliquer concrètement l’intégration d’une API de paiement (FedaPay), une compétence essentielle dans le développement d’applications web modernes.
